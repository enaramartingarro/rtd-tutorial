Welcome to Lumache's documentation!
===================================

This script allows the user predict sales of a cluster or market.
OTC and DE sales are combined.
Insurer's preferences are considered when when appropriate.

The input: the sales (DE+OTC) of the market, the insurance 
companies preferences and the Dispense Events of the market.

.. note::

   This project is under active development.
